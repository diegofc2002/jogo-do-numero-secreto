# 🎯 Jogo do Número Secreto

Um jogo de adivinhação desenvolvido com HTML, CSS e JavaScript, onde o objetivo é descobrir o número secreto gerado aleatoriamente pelo sistema.

🔗 **[Jogar agora](https://jogo-do-numero-secreto-henna-omega.vercel.app)**

---

## 📋 Sobre o Projeto

O **Jogo do Número Secreto** é uma aplicação web em que o jogador tenta adivinhar um número secreto escolhido aleatoriamente. A cada tentativa, o jogo fornece dicas indicando se o número secreto é maior ou menor do que o chute feito. O jogo conta com uma opção de reinício para uma nova partida.

---

## ✨ Funcionalidades

- 🔢 Geração aleatória de um número secreto
- 💡 Dicas após cada tentativa (maior ou menor)
- 🔁 Botão de "Novo jogo" para reiniciar a partida
- 🖼️ Interface visual com imagem ilustrativa

## 🚀 Como Executar Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/diegofc2002/jogo-do-numero-secreto.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd jogo-do-numero-secreto
   ```

3. **Abra o arquivo `index.html` no navegador:**
   - Clique duas vezes no arquivo, ou
   - Use uma extensão como **Live Server** no VS Code para uma melhor experiência.

> Nenhuma dependência ou instalação adicional é necessária!

---

## 🎮 Como Jogar

1. Acesse o jogo pelo navegador.
2. Um número secreto é gerado automaticamente pelo sistema.
3. Digite um número no campo de entrada e clique em **"Chutar"**.
4. O jogo indicará se o número secreto é **maior** ou **menor** do que o seu chute.
5. Continue tentando até acertar!
6. Ao acertar, clique em **"Novo jogo"** para jogar novamente.
